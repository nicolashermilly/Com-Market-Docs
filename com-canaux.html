1 À quoi servent les canaux ?

Un canal de diffusion représente un point de contact par lequel votre communication atteint votre audience. Ils structurent ensuite vos contenus, vos sources de leads et vos statistiques.

Définition

Chaque canal porte un libellé (affiché partout dans l'application), un code technique unique (utilisé par les intégrations) et un type qui le classe dans une grande famille. Les canaux que vous définissez ici alimentent les menus déroulants des autres écrans (contenus, sources de leads).

Les 5 types disponibles

  • Blog — articles de fond, SEO, pages piliers.
  • Social — LinkedIn, Instagram, X, et autres réseaux.
  • Email — newsletters, séquences, emailing one-shot.
  • Presse — communiqués, relations médias.
  • Autre — tout canal qui n'entre pas dans les catégories ci-dessus.
2 Gérer ses canaux

Tout se passe sur une seule page : Paramètres › Canaux de diffusion.

2.1 Ajouter un canal
  • Remplissez la dernière ligne du tableau : Libellé, Code, Type.
  • Cliquez sur Ajouter le canal : il apparaît en bas de la liste.
  • Le code doit être unique ; un doublon est refusé.
Astuce : gardez des codes courts et stables (linkedin, newsletter) — ils servent de clé aux intégrations et ne devraient pas changer.
2.2 Modifier un canal
  • Cliquez directement dans une cellule (libellé, code ou type) et éditez.
  • La modification est enregistrée automatiquement dès que vous quittez le champ.
2.3 Réordonner par glisser-déposer
  • Saisissez la poignée à gauche d'une ligne et faites-la glisser.
  • Le nouvel ordre est mémorisé ; il persiste après rechargement de la page.
Bon à savoir : l'ordre que vous définissez ici est celui repris dans les listes déroulantes des autres écrans.
2.4 Désactiver / réactiver
  • Le bouton Désactiver retire un canal de l'usage courant sans le supprimer.
  • Cochez Afficher les désactivés pour les revoir et les réactiver.
Important : il n'existe pas de suppression définitive. C'est volontaire : l'historique (contenus, leads rattachés à un canal) reste toujours cohérent.

4 Questions fréquentes
Puis-je supprimer définitivement un canal ?
Non, et c'est intentionnel. Un canal peut être rattaché à des contenus ou des sources de leads ; le supprimer casserait cet historique. Utilisez Désactiver : le canal disparaît de l'usage courant tout en restant référençable.
Le code d'un canal peut-il changer ?
Techniquement oui, mais ce n'est pas recommandé : le code sert de clé stable. Préférez modifier le libellé (affichage) et laisser le code intact.
Mes canaux sont-ils partagés entre clients ?
Non. Chaque espace client (tenant) possède sa propre liste de canaux. Ce que vous définissez ici ne concerne que votre organisation.

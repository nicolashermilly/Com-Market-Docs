1 À quoi servent les sources ?

Une source de lead décrit l'origine d'un contact. Bien renseignée, elle vous dit quels efforts d'acquisition portent leurs fruits.

Définition

Chaque source porte un libellé (affiché partout dans l'application), un code technique unique et, en option, un canal de rattachement. Les sources que vous définissez ici alimentent le champ « source » des leads dans le pipeline.

Le rattachement à un canal

Une source peut être reliée à un canal de diffusion — par exemple, la source « Formulaire newsletter » rattachée au canal « Email ». Ce lien est facultatif : une source peut très bien rester sans canal. Quand un canal est choisi, son libellé s'affiche à côté de la source.

2 Gérer ses sources

Tout se passe sur la page Paramètres › Sources de leads.

2.1 Ajouter une source
  • Renseignez Libellé et Code sur la dernière ligne, puis validez.
  • Le code doit être unique ; un doublon est refusé.
  • Choisissez éventuellement un canal dans la liste déroulante (vous pouvez laisser vide).
Astuce : gardez des codes courts et stables (recommandation, salon) — ils restent l'identifiant de la source même si vous renommez le libellé.
2.2 Rattacher à un canal
  • Dans la colonne Canal, ouvrez la liste et choisissez un canal existant.
  • La liste est alimentée par vos canaux de diffusion ; tenez-les à jour au préalable.
  • Laissez sur « aucun » si la source ne correspond à aucun canal.
Bon à savoir : le lien pointe vers le canal par une référence ; renommer le libellé d'un canal n'affecte pas le rattachement.
2.3 Modifier, réordonner
  • Cliquez dans une cellule pour éditer ; la modification s'enregistre automatiquement.
  • Réordonnez par glisser-déposer via la poignée à gauche ; l'ordre persiste après rechargement.
2.4 Désactiver / réactiver
  • Le bouton Désactiver retire une source de l'usage courant sans la supprimer.
  • Cochez Afficher les désactivés pour la revoir et la réactiver.
Important : pas de suppression définitive. C'est volontaire : les leads déjà rattachés à une source restent cohérents.

4 Questions fréquentes
Une source doit-elle obligatoirement avoir un canal ?
Non. Le rattachement à un canal est facultatif. Une source « Bouche-à-oreille », par exemple, peut très bien rester sans canal.
Quelle différence entre une source et un canal ?
Un canal est un moyen de diffusion (blog, email, presse...). Une source est l'origine précise d'un contact, parfois plus fine qu'un canal : plusieurs sources peuvent pointer vers le même canal.
Puis-je supprimer définitivement une source ?
Non. Utilisez Désactiver : la source disparaît de l'usage courant tout en préservant les leads qui y sont rattachés.

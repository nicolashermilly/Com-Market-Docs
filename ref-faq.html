Questions fréquentes

Les réponses aux questions les plus courantes sur Cockpit'Gest, regroupées par thème.

Référence FAQ
1 Général
Par où commencer ?
Suivez le guide de paramétrage : entreprise, charte, canaux, verticales. Puis explorez le Cockpit et le Pipeline.
Mes données sont-elles isolées des autres clients ?
Oui. Chaque espace (tenant) est séparé ; vos données ne sont visibles que par votre organisation. Le badge en haut indique l'espace courant.
2 Paramétrage
Puis-je supprimer définitivement un élément ?
Non : Cockpit'Gest privilégie la désactivation. Un élément retiré reste conservé (pour la cohérence de l'historique) et peut être réactivé.
Comment réordonner une liste ?
Par glisser-déposer : saisissez la poignée à gauche d'une ligne et déplacez-la. L'ordre est mémorisé et persiste après rechargement.
À quoi sert le code d'un élément ?
Le code est un identifiant technique stable, distinct du libellé affiché. Modifiez librement le libellé ; évitez de changer le code.
3 Apparence
Comment changer les couleurs de l'interface ?
Dans Charte graphique : couleurs, logo, nom d'application, avec un aperçu live.
La couleur des documents suit-elle celle de l'interface ?
Non, elles sont indépendantes. La couleur d'accent des documents se règle dans Éditions & modèles.
